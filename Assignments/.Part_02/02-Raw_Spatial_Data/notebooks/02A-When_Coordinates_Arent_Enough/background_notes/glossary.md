# Glossary — When Coordinates Aren’t Enough

**Coordinate**  
A numeric value representing position within a defined reference system.

**(x, y) Pair**  
Two numeric values that indicate position relative to an origin and axes.

**Reference System**  
An agreed-upon framework that defines how coordinates map to real space.

**Origin (0,0)**  
The point from which coordinates are measured.

**Orientation**  
The direction in which coordinate values increase.

**Units**  
The measurement system used by coordinates (degrees, meters, pixels, etc.).

**Location**  
A position that has meaning because it is defined within a reference system.

**Context**  
Information that explains what a coordinate represents and why it exists.

**Silent Failure**  
An error that produces plausible-looking output without warning.

**Geographic Coordinates**  
Coordinates based on latitude and longitude that reference positions on Earth.
